### msx-go
=====
a set of types for interacting with the msx (media station x) api

all types refer to the official documentation: https://msx.benzac.de/wiki/index.php?title=Welcome

## Getting Started

### Installing

To start using `msx-go`, install Go and run `go get`:
```sh
$ go get github/mount-grabber/msx-go@latest
```
